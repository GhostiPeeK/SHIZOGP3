\# SHIZOGP Bot



Простой Telegram бот для тестирования на BotHost.



\## Команды

\- /start - Начать работу

\- /help - Помощь

\- /info - Информация

\- /balance - Баланс

\- /referral - Реферальная ссылка



\## Настройка на BotHost

1\. Добавь переменную окружения `BOT\_TOKEN`

2\. Команда запуска: `python bot/main.py`
